Quick Start

01
Clone the flake
$ git clone https://github.com/ArcoNightshade/LyonOS-V2 ~/.lyonos $ rm -rf ~/.lyonos/.git
02
Copy hardware config
$ cp /etc/nixos/hardware-configuration.nix ~/.lyonos/profile/workstation/hardware.nix
03
Set username in flake.nix
$ nano ~/.lyonos/flake.nix # set account.name and timeZone
04
Configure drive & password in compose.nix
$ mkpasswd -m sha-512 <password> # generate hash $ nano ~/.lyonos/compose.nix # paste hash, set drive type
Drive options at the bottom: /dev/vda (VM), /dev/sda (HDD), /dev/nvme0n1 (SSD — default)
05
Build & boot
$ sudo nixos-rebuild boot --flake ~/.lyonos#workstation
Use boot to build next generation and apply on reboot. Use switch to apply immediately.

System Profiles

💻
WORKSTATION
Primary laptop / daily driver
The main profile. Hardware config lives at profile/workstation/hardware.nix — replace with your own from /etc/nixos.
sudo nixos-rebuild switch --flake ~/.lyonos#workstation
🖥
DESKTOP
Desktop machine
Separate hardware profile for the desktop. Same flake, different hardware config and configuration.nix. Use the same setup flow, targeting #desktop.
sudo nixos-rebuild switch --flake ~/.lyonos#desktop
📦
ISO
Virtual machines
Stripped-down profile for VMs and ISO builds. Drive defaults to /dev/vda. Useful for testing config changes before applying to bare metal.
sudo nixos-rebuild switch --flake ~/.lyonos#iso

Common Commands

Rebuild
$sudo nixos-rebuild switch --flake ~/.lyonos#workstation
Applies immediately. Use boot to defer until reboot. dry-run evaluates without building. test switches without writing to boot menu.
Rollback
$sudo nixos-rebuild switch --flake ~/.lyonos#workstation --rollback
Reverts to the previous generation. Generations accumulate until garbage collected.
Update
$sudo nix flake update --flake ~/.lyonos#workstation
Updates flake.lock to latest commits of all inputs. Rebuild after to apply.
Garbage collect
$sudo nix-collect-garbage
Deletes all old generations. nh auto-cleans generations older than 4 days, keeping 3. Run manually if running low on space.
Temp shell
$nix-shell -p cowsay
Installs software into a temporary shell. Discarded when the shell closes. No system changes.
Rebuild modes
switch  apply now
boot    apply on next reboot
dry-run evaluate only
test    switch without writing bootloader
